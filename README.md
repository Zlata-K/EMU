## Controls:

Move - AWSD / Arrows <br />
Interact with environment - E <br />
1 / 2 - Select primary or secondary weapon <br />
Shoot - K <br />
Throw grenades - G <br />
Reload - R <br />
Jump - Spacebar <br />
Roll - Shift <br />
Return to the main menu - Escape <br />
